## Miconic-SIMPLE

*   Origin: Jana Koehler. Used in the AIPS-2000 competition.
*   Adaptions: None.
*   Description: Typed ADL domain using conditional effects. Like above,
    but without any additional constraints. The conditional effects make
    sure that all waiting passengers get in or out.
*   Parameters:
    *   -f number of floors
    *   -p number of passengers
    *   -r random seed
*   Generation: Original generator used in the AIPS-2000 competition.
    Simply distribute origin and destination floors at random.
