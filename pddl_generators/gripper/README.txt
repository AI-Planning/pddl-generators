## Gripper

*   Origin: Jana Koehler. Used in the AIPS-1998 competition.
*   Adaptions: None.
*   Description: Untyped STRIPS domain. Given a robot with two gripper
    hands, transport a number of balls from a room A to another room B.
*   Parameters: -n number of balls.
*   Generation: No randomization. Place all balls in room A and require
    them to be in B instead.
